## Ellipsoidal Regions
#### Circles
`centerX`;`centerY`;`radius`
#### Ellipses
`centerX`;`centerY`;`radius1`;`radius2`
#### Examples
```csv
-3.5;4;3
5.0;-2.77;6;2
```

## Polygonal Regions
#### Rectangles
`point1X`;`point1Y`

`point2X`;`point2Y`
#### Triangles
`point1X`;`point1Y`

`point2X`;`point2Y`

`point3X`;`point3Y`
#### Other polygons
`point1X`;`point1Y`

`point2X`;`point2Y`

...

`pointnX`;`pointnY`
#### Examples
```csv
26.53;18.12
25.3;22.1
29.4;25.3
32.2;22.1
31.0;18.12
```
